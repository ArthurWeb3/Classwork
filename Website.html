BUS 118W, or Web Based Computing is an upper-division Management Information Systems (MIS) elective. The course focuses on web development fundamentals, such as HTML, CSS, Javascript, React, Bootstrap, and so on. Each module in the textbook guides students step by step on how to properly create, develop, and publish a website. Visual Studio Code or any code editor is required and part of course completion. Students also learn how to use platforms like GitHub to create and share repositories amongst fellow students and online collaborators.
